# 🚀 Safora QA Automation Project

## 📌 Overview
This project was developed as part of a **QA Engineering Intern assignment** for testing the Safora website.

It includes:
- Manual Test Case Document (Part 1)
- Selenium WebDriver Automation (Part 2)

---

## 🧪 Project Scope

### Part 1 – Manual Testing
A structured test case document was created after exploring the Safora website.

### Covered Areas:
- Homepage testing
- Navigation testing
- Contact Us form testing
- Login functionality testing
- Pricing page verification
- Footer link validation
- Accessibility testing
- Performance testing
- Cross-browser testing
- Security testing (basic level)

---

### Part 2 – Automation Testing

Automated test cases were developed for the **Contact Us form** using Selenium WebDriver.

### Tools Used:
- Java 21
- Selenium WebDriver 4.18.1
- TestNG 7.9.0
- Maven
- WebDriverManager
- Eclipse IDE

### Automated Scenarios:
- Valid form submission
- Empty form validation
- Missing email validation
- Invalid email format validation
- UI interaction verification
